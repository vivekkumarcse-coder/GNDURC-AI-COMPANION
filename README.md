# 📖 About the Project

GNDURC AI Companion is an intelligent information chatbot developed using **Python** and **Tkinter** for students of **Guru Nanak Dev University Regional Campus (GNDURC), Gurdaspur**.

The chatbot serves as a virtual campus assistant by providing instant access to essential university information through a simple graphical interface. Instead of manually searching notices, documents, or websites, users can quickly retrieve information related to admissions, courses, fees, scholarships, placements, hostel facilities, and more.

The project demonstrates the practical implementation of Python GUI programming, data organization, and chatbot-based information retrieval in an educational environment.

---

# 🎯 Project Objectives

The objectives of this project are:

- 📌 Provide quick access to university information.
- 📌 Reduce the time spent searching for college-related details.
- 📌 Offer a simple and user-friendly graphical interface.
- 📌 Organize academic information in a structured format.
- 📌 Improve communication between students and the institution.
- 📌 Demonstrate the use of Python in GUI application development.

# ⚙ How It Works

1️⃣ User launches the chatbot.

2️⃣ User enters or selects a query.

3️⃣ The chatbot searches the stored data.

4️⃣ Relevant information is retrieved.

5️⃣ Images are displayed (if available).

6️⃣ The response is shown in the chat window.

7️⃣ Invalid queries display an appropriate message.
